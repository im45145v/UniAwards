## Performance Optimization

### Before (N+1 Query Pattern)
```
Query 1: SELECT * FROM polls
Query 2: SELECT * FROM nominations WHERE poll_id = 'poll1'
Query 3: SELECT * FROM votes WHERE poll_id = 'poll1'
Query 4: SELECT * FROM nominations WHERE poll_id = 'poll2'
Query 5: SELECT * FROM votes WHERE poll_id = 'poll2'
...
Total: 1 + (2 Ã— N polls) queries
```

### After (Bulk Query Pattern)
```
Query 1: SELECT * FROM polls
Query 2: SELECT * FROM nominations WHERE poll_id IN [...]
Query 3: SELECT * FROM votes WHERE poll_id IN [...]

Total: 3 queries (regardless of poll count)
```
